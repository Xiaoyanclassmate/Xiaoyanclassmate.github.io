linux笔记

2023/5/29

管理文件类型

ls 查看当前目录或文件

-a ：显示当前目录下的所有文件和目录
-A ：显示所有文件和目录，但不显示当前目录“.”和上层目录“..”
-d：显示目录本身而不是目录下的内容。
-l：使用详细格式列表，显示类型、权限、所属用户、组等信息
-c：以更改时间排序，显示文件和目录。
-s：显示文件和目录的大小，以 blocks 为单位。

pwd 查看当前文件

cd 切换目录

查看文件内容

cat 可以滚屏显示文件的内容

more和less 分屏查看，摁Q退出

head和tail 查看一个文件开头或者结尾而非文件的全部内容

grep 强大的文本搜索工具，并把匹配的好好打印出来

touch 创建文件

mkdir 创建目录

rmdir 删除目录

-p:删除指定目录后，若该目录的上层目录已变成空目录，则将其一并删除

rm 删除文件或目录

-f:强制删除文件或目录，不进行提示。
-i:删除既有文件或目录之前先询问用户。
-R:递归处理，将指定目录下的所有文件及子目录一并处理。

cp 复制文件或目录

格式： cp 参数 源文件 目标文件
-f:强行复制文件或目录，无论目标文件或目录是否已存在。
-l:对源文件建立硬连接，而非复制文件。
-S:对源文件建立符号连接，而非复制文件。
-R:递归处理，将指定目录下的所有文件与子目录一并处理。

mv 用于对文件或目录进行移动或改名

格式： mv 参数 源文件或目录 目标文件或目录
-f:若目标文件(目录)与现有的文件(目录)重复，则直接覆盖现有的文件(目录)
-i:覆盖前先询问用户。

hello*……*： 指目录里关于前面有hello的名字包括全部

硬链接： 两个文件名指向的是硬盘上的同一块储存空间，对任何一个文件的修改将影响到另一个文件
软连接： (符号链接），类似于快捷方式

ln 创建链接:
格式:ln 参数 源文件或目录 链接名称

-s:对源文件建立符号链接，而非硬链接。

find 查找文件
格式:find 路径 匹配表达式

权限的分类
Linux 文件的权限有如下几种:

  1. r(read):读取，对文件来说是读取内容，对目录来说是浏览目录内容。
  2. w(write):写入，对文件来说是修改文件内容，对目录来说是删除和修改目录内文件。
  3. x(execute):执行，对文件来说是执行文件，对目录来说是进入目录。
  4. “_”:表示不具有该项权限。

chmod 更改文件以及目录的权限
格式:chmod 选项 对权限的设定 文件或者目录
选项: g+x,o+w 给文件同组用户，其他用户添加写权限
o-rw 给文件其他用户删除读写权限
a+x 给文件所有者，用户组和其他用户添加执行权限
u=rwx,g=rw,o=r 设定文件所有者读写执行权限、用户组读写权限,以及其他用户只读权限

chowm 改变一个文件或者目录的所属用户(既拥有者)
格式:chown 用户名 文件或者目录
-R表示递归处理,当操作项是目录的时候,表示把目录中所有的文件以及子目录的拥有者

chgrp 更改文件目录所属组
格式:chgrp 组名 文件或者目录

umask 更改默认权限
格式:umask 002

vi编辑器
vi new.txt 进入编辑器
vipw 直接查看etc/passwd
vim 是vi的升级版

useradd 和 adduser 添加新用户
passwd 管理用户口令
usermod 修改用户信息
userdel 删除用户
su 切换登录用户

实训四
fdisk -l 查看磁盘信息
fdisk常用命令说明如下:
d:删除一个分区。
l: 列出已知分区类型。
m:列出帮助菜单。
n:添加一个新分区。
p:显示分区表。
q:退出不保存
t:更改分区类型ID(如swap分区ID是82，ext4分区ID是83）
w:写入并退出

mkfs.ext4➕主分区
ext4:扩展日志式文件系统，还有123的版本
mke2fs 默认创建的是ext2文件系统，添加-j参数才可以创建ext3文件系统

mount 挂载命令
mount 选项 设备名称 挂载点目录
-t 指定挂载的文件系统类型
-r 以只读方式挂载

df 查询挂载
-m MB的单位
-Th 详细的显示内存信息

umount 卸载挂载的文件系统

让分区支持磁盘配额，编辑etc/fstab文件，既要用户配额，又要支持组配额，需在磁盘上添加”usrquota”(用户配额），以及”grpquota”(用户组配额)，完成后重启

quotacheck 检查添加了配额项的磁盘，并为磁盘建立一个当前磁盘用量表，以下命令一般全用
-a 扫描已添加usrquota或grpquota的分区
-g 检查文件系统中文件和目录的数目，创建aquota.group文件
-u 检查文件系统中文件和目录的数目，创建aquota.user文件
-v 显示命令执行过程

edquota 设置用户和组的配额
edquota 选项 用户名｜组名
-u 设置用户磁盘配额，后接用户名
-g 设置组磁盘配额，后接组名

quotaon 启动磁盘配额
quotaoff 关闭磁盘配额
