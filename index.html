华为通信数据
2023/11/26

1、熟悉VRP基本操作

display version 显示系统软件版本信息及硬件信息

system-view 切换系统视图

quit 返回上一视图

interface + 接口 进入接口视图

return或Ctrl+z 直接退回用户视图

在命令后输入？，可以获得后面的命令提示

sysname [名称] 修改名称

clock datetime 时:分:秒 年-月-日 设置路由器时钟

clock timezone BJ add 08:00:00 设置所在地区及时间

header login information “hello” 设置登录的标题为hello

header shell information “hello” 设置登录成功后的标题文本

display 系列命令可查看基本信息或运行状态

display interface GigabitEthernet 0/0/0 命令查看路由器GE 0/0/0接口的状态信息

1.3、熟悉常用的IP相关命令

save 保存信息

display ip interface brief 查看接口与IP相关信息

display ip routing-table 查看IPV4路由表的信息

ip address 配置接口ip地址、子网掩码

1.4、配置通过Telnet登陆系统

ip route-static 0.0.0.0 0.0.0.0 10.1.1.254 配置网关

user-interface vty 0 4 进入虚拟终端,VTY用户界面

authentication-mode password 进入设置密码模式

telnet [主机IP地址] 使用telnet连接主机

display users 登录成功后查看登录的用户信息

set authentication password cipher huawei 设置密码及加密

user privilege level 1 配置用户级别为 1(监控级)

aaa 进去aaa视图

local-user admin password cipher hello privilege level 3 在aaa视图配置本地用户民 admin和密文密码hello并将用户级别修改为3(管理级)

local-user admin service-type telnet 配置该用户接入类型为Telnet

user-interface vty 0 4
authentication-mode aaa 进入VTY用户界面视图下，将认证模式改成AAA
