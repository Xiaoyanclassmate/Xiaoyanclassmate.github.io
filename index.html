一、Markdown简介

  Markdown 是一种轻量级标记语言，创始人为约翰·格鲁伯（John Gruber）。它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的 XHTML（或者HTML）文档。Markdown支持HTML语法。

二、Markdown常用语法

2.1 文章目录

2.1.1 TOC方法

  TOC全称为Table of Content，自动列出全部标题。在文章开头使用[TOC]即可自动生成文章目录。

2.1.2 手动生成目录

  将「列表」和「页内超链接」相结合，但有些Markdown编辑器不支持，有点类似于HTML的锚点。
语法格式：

目录名
规则：

标题链接格式为： 一个#+被链接标题。
标题链接中不能出现大写字母，大写字母用小写字母代替。
标题链接中不能出现空格 ，空格 用-代替。
目录排布由有序列表或无序列表控制
跳转与目录名无关，标题和标题链接符合规则即可
例子（只在VsCode上测试可行）：

// 目录
[1 一级标题] (#1-一级标题)

[1.1 二级标题] (#11-二级标题)

[1.2 二级标题] (#12-二级标题-title)

[1.3 二级标题] (#13-二-级-标-题-space-test)

// 标题
#1 一级标题
##1.1 二级标题
##1.2 二级标题 Title
##1.3 二 级 标 题 Space Test
以上记得在 # 后面空行

linux笔记
2023/5/29

管理文件类型

ls 查看当前目录或文件

-a ：显示当前目录下的所有文件和目录
-A ：显示所有文件和目录，但不显示当前目录“.”和上层目录“..”
-d：显示目录本身而不是目录下的内容。
-l：使用详细格式列表，显示类型、权限、所属用户、组等信息
-c：以更改时间排序，显示文件和目录。
-s：显示文件和目录的大小，以 blocks 为单位。

pwd 查看当前文件

cd 切换目录

查看文件内容

cat 可以滚屏显示文件的内容

more和less 分屏查看，摁Q退出

head和tail 查看一个文件开头或者结尾而非文件的全部内容

grep 强大的文本搜索工具，并把匹配的好好打印出来

touch 创建文件

mkdir 创建目录

rmdir 删除目录

-p:删除指定目录后，若该目录的上层目录已变成空目录，则将其一并删除

rm 删除文件或目录

-f:强制删除文件或目录，不进行提示。
-i:删除既有文件或目录之前先询问用户。
-R:递归处理，将指定目录下的所有文件及子目录一并处理。

cp 复制文件或目录

格式： cp 参数 源文件 目标文件
-f:强行复制文件或目录，无论目标文件或目录是否已存在。
-l:对源文件建立硬连接，而非复制文件。
-S:对源文件建立符号连接，而非复制文件。
-R:递归处理，将指定目录下的所有文件与子目录一并处理。

mv 用于对文件或目录进行移动或改名

格式： mv 参数 源文件或目录 目标文件或目录
-f:若目标文件(目录)与现有的文件(目录)重复，则直接覆盖现有的文件(目录)
-i:覆盖前先询问用户。

hello*……*： 指目录里关于前面有hello的名字包括全部

硬链接： 两个文件名指向的是硬盘上的同一块储存空间，对任何一个文件的修改将影响到另一个文件
软连接： (符号链接），类似于快捷方式

ln 创建链接:
格式:ln 参数 源文件或目录 链接名称

-s:对源文件建立符号链接，而非硬链接。

find 查找文件
格式:find 路径 匹配表达式

权限的分类
Linux 文件的权限有如下几种:

  1. r(read):读取，对文件来说是读取内容，对目录来说是浏览目录内容。
  2. w(write):写入，对文件来说是修改文件内容，对目录来说是删除和修改目录内文件。
  3. x(execute):执行，对文件来说是执行文件，对目录来说是进入目录。
  4. “_”:表示不具有该项权限。

chmod 更改文件以及目录的权限
格式:chmod 选项 对权限的设定 文件或者目录
选项: g+x,o+w 给文件同组用户，其他用户添加写权限
o-rw 给文件其他用户删除读写权限
a+x 给文件所有者，用户组和其他用户添加执行权限
u=rwx,g=rw,o=r 设定文件所有者读写执行权限、用户组读写权限,以及其他用户只读权限

chowm 改变一个文件或者目录的所属用户(既拥有者)
格式:chown 用户名 文件或者目录
-R表示递归处理,当操作项是目录的时候,表示把目录中所有的文件以及子目录的拥有者

chgrp 更改文件目录所属组
格式:chgrp 组名 文件或者目录

umask 更改默认权限
格式:umask 002

vi编辑器
vi new.txt 进入编辑器
vipw 直接查看etc/passwd
vim 是vi的升级版

ext4:扩展日志式文件系统，还有123的版本

mount 挂载命令
mount 选项 设备名称 挂载点目录
-t 指定挂载的文件系统类型

-Th 详细的显示内存信息

-a 扫描已添加usrquota或grpquota的分区

-u 检查文件系统中文件和目录的数目，创建aquota.user文件
-v 显示命令执行过程

edquota 选项 用户名｜组名
-u 设置用户磁盘配额，后接用户名
